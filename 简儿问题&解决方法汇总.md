# 简儿问题 & 解决方法汇总

这里汇总了简儿QQ机器人的常见问题和解决方法，帮助您快速排查和解决使用过程中遇到的问题。

## 1、Lagrange.OneBot 登录失败 提示Signer Server returned a 521
这是由于Lagrange.Onebot官方关闭了签名服务器，导致无法正常登录，我们建议改用[NapCat](./NapCatQQ使用教程.html) 如果您仍因为各种奇妙的原因(如性能限制，格式不兼容)无法使用NapCat 可使用我们提供的SignServer 打开appsettings.json 将里面的SignServerUrl改成`https://sign.ae3803.icu/api/sign/41857/`(主，建议使用)，如果提示code 45 请尝试将41857改成42941临时登录，之后改回来

请输入python --version 查看python版本是否是3.12+
