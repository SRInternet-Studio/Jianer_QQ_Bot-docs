# 第四步：启动并使用

恭喜你走到了最后一步！现在你已经下载了简儿、安装好了运行环境、完成了配置，只差一步就能让你的机器人在 QQ 群里正式上线了。

## 启动顺序

::: warning 重要：请严格按顺序操作
必须先启动 NapCatQQ（协议端），**再**启动简儿（机器人主程序）。如果反过来，简儿会因为找不到协议端而连接失败。
:::

### 第 1 步：启动 NapCatQQ

确保你的 NapCatQQ 已经正确安装和配置（参考 [NapCatQQ 使用教程](/NapCatQQ使用教程)）。

启动 NapCatQQ 后，确认以下几件事：

- ✅ NapCatQQ 已经成功启动
- ✅ 你的机器人 QQ 号已经扫码登录成功
- ✅ WebUI 中的 WebSocket 服务器正在运行（端口与 `config.json` 中一致）

### 第 2 步：启动简儿主程序

你有两种方式启动简儿：

**方式一：通过设置向导启动**

如果你之前使用 `SetupWizard.pyw` 完成了配置，可以直接在设置向导中启动：

1. 运行 `python SetupWizard.pyw`
2. 打开 **"核对并应用设置"** 页面
3. 点击 **"启动吧，我的简儿！"** 按钮

**方式二：通过命令行启动**

在简儿的程序根目录中打开终端，运行：

```shell
python main.py
```

### 判断是否启动成功

启动后，观察终端中的日志输出。当你看到类似以下内容时，说明简儿已经成功连接到了协议端：

```
ℹ️ INFO 成功建立连接
```

看到这条消息后，你就可以去 QQ 群里和你的机器人互动了！🎉

## 启动后做什么

### 试着跟机器人说句话

在你的机器人所在的 QQ 群里，发送以下消息试试：

```
~帮助
```

（这里的 `~` 是你在 `config.json` 中设置的 `reminder` 触发符号。如果你改成了其他符号，请用你设置的那个。）

如果机器人回复了帮助信息，说明一切就绪！

你也可以直接 **@机器人** 来查看帮助内容。

### 让机器人拥有 AI 能力

现在你的机器人可以正常工作了，但如果你想让它拥有 AI 对话能力（支持 ChatGPT、Gemini、DeepSeek），需要额外配置 API Key。

👉 [配置 AI 功能](/Configuring-AI-Functions)

## 如果启动失败了怎么办

### 提示"连接失败"或"无法连接到 WebSocket"

这说明简儿无法连接到 NapCatQQ。请检查：

1. NapCatQQ 是否已经启动并成功登录？
2. NapCatQQ 的 WebSocket 服务器是否已经开启？
3. `config.json` 中的 `Connection.host` 和 `Connection.port` 是否与 NapCatQQ WebUI 中的设置一致？

### 提示 ModuleNotFoundError

这说明某个依赖库没有安装。根据报错信息手动安装即可：

```shell
pip install 缺失的库名
```

### 提示 SyntaxError

检查你的 Python 版本是否满足要求（NEXT 3 需要 Python 3.12+）：

```shell
python --version
```
